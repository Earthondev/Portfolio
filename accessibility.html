Accessibility Statement

คำแถลงการเข้าถึงได้

Last updated: January 1, 2025

ความมุ่งมั่นของเรา

ตั้งใจให้เข้าถึงได้ตาม WCAG 2.1 AA: คอนทราสต์ดี อ่านง่าย พิสูจน์ด้วย Pa11y/Lighthouse

เราเชื่อว่าทุกคนควรสามารถเข้าถึงและใช้งานเว็บไซต์ได้อย่างเท่าเทียมกัน โดยไม่คำนึงถึงความสามารถทางกายภาพหรือเทคโนโลยีที่ใช้

มาตรฐานการเข้าถึงได้

เว็บไซต์นี้ได้รับการออกแบบและพัฒนาให้เป็นไปตามมาตรฐาน Web Content Accessibility Guidelines (WCAG) 2.1 ระดับ AA

หลักการที่เราใช้:

  • Perceivable (รับรู้ได้): เนื้อหาและข้อมูลสามารถรับรู้ได้ผ่านประสาทสัมผัสต่างๆ
  • Operable (ใช้งานได้): สามารถนำทางและใช้งานได้ด้วยวิธีต่างๆ
  • Understandable (เข้าใจได้): ข้อมูลและวิธีการใช้งานเข้าใจง่าย
  • Robust (แข็งแกร่ง): สามารถใช้งานได้กับเทคโนโลยีช่วยเหลือต่างๆ

คุณสมบัติการเข้าถึงได้

การนำทาง:

  • Skip links สำหรับข้ามไปยังเนื้อหาหลัก
  • โครงสร้างหัวเรื่องที่ชัดเจน (H1, H2, H3)
  • Navigation ที่ใช้งานได้ด้วยคีย์บอร์ด
  • Breadcrumb navigation

การแสดงผล:

  • คอนทราสต์สีที่เพียงพอ (4.5:1 สำหรับข้อความปกติ)
  • Dark/Light mode toggle
  • Responsive design สำหรับทุกขนาดหน้าจอ
  • Font size ที่อ่านง่าย

เนื้อหา:

  • Alt text สำหรับรูปภาพทั้งหมด
  • ARIA labels และ roles ที่เหมาะสม
  • Focus indicators ที่ชัดเจน
  • Semantic HTML elements

การทดสอบ

เราใช้เครื่องมือต่างๆ เพื่อทดสอบการเข้าถึงได้:

  • Lighthouse: ทดสอบคะแนน Accessibility
  • Pa11y: ทดสอบ WCAG compliance
  • Manual testing: ทดสอบด้วยคีย์บอร์ดและ screen reader
  • Color contrast: ทดสอบอัตราส่วนคอนทราสต์สี

เทคโนโลยีช่วยเหลือที่รองรับ

  • Screen readers (NVDA, JAWS, VoiceOver)
  • Voice control software
  • Switch navigation
  • Magnification software
  • Keyboard-only navigation

การปรับปรุงอย่างต่อเนื่อง

เราให้ความสำคัญกับการปรับปรุงการเข้าถึงได้อย่างต่อเนื่อง โดย:

  • ทดสอบเป็นประจำด้วยเครื่องมืออัตโนมัติ
  • รับฟังข้อเสนอแนะจากผู้ใช้
  • อัปเดตตามมาตรฐานใหม่ๆ
  • ฝึกอบรมทีมพัฒนาเรื่อง accessibility
