HaaNaiHang

Mall & Store Finder + Admin Panel

2025 Web App Product Designer, Frontend, Tester
React TypeScript Vite Tailwind Firebase Netlify

Project Overview

เว็บช่วยค้นหาห้าง/ร้านค้าใกล้ตัว พร้อมแผงผู้ดูแลในการเพิ่ม/แก้ไขข้อมูลศูนย์การค้าและร้านค้าแบบเป็นระบบ

Problem

ผู้ใช้ต้องการค้นหาห้างสรรพสินค้าและร้านค้าใกล้ตัว แต่ข้อมูลไม่เป็นระบบและไม่มีการจัดการที่เหมาะสม

Solution

สร้างเว็บแอปพลิเคชันที่ช่วยค้นหาห้าง/ร้านค้าได้อย่างรวดเร็ว พร้อมระบบ Admin Panel สำหรับจัดการข้อมูล

Key Features

  • ค้นหาห้าง/ร้านได้ใน 1-2 คลิก - รองรับมือถือเต็มรูปแบบ
  • Admin ลดเวลาบันทึกข้อมูล ~35-45% - ผ่านฟอร์มที่มี validation + autosave
  • A11y ≥ 95 | Lighthouse ≥ 95 - โหลด < 2s (WebP + lazy + preconnect)
  • QA Gates - required-fields, range check, duplicate guard (Firestore rules)
  • Analytics - GA4 events (theme_toggle, project_view, search_query, gallery_nav)
  • Google Analytics 4 - ติดตามการใช้งานจริง → เห็น Top search keywords/ตำแหน่งยอดนิยม
  • Firebase Security Rules - Input validation ป้องกันข้อมูลผิดพลาด

Key Screens

Home Page

หน้าแรก: ปุ่มระบุตำแหน่ง + ช่องค้นหา + การ์ดห้าง

Admin Dashboard

Admin: ตารางรายการศูนย์การค้าและสรุปข้อมูลสำคัญ

Store Management

Admin: ฟอร์มเพิ่มร้านค้า—เลือกห้าง, ชั้น, หมวดหมู่, เวลาเปิดปิด

Search Results

หน้าค้นหา—ผลลัพธ์ห้างหลายรายการพร้อมระยะทาง

  Google Analytics 4

Key Features Implementation

  • Geolocation API: ระบุตำแหน่งปัจจุบันของผู้ใช้
  • Firestore Queries: ค้นหาห้าง/ร้านตามตำแหน่งและระยะทาง
  • React Context: จัดการ state ของ theme และ user preferences
  • Custom Hooks: useGeolocation, useFirestore, useAnalytics
  • Form Validation: Yup schema validation สำหรับ Admin forms
  • Image Optimization: WebP format + lazy loading
  • PWA Features: Service Worker + Offline support
